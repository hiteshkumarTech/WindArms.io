# WindArms — Project Overview

A browser-based multiplayer FPS, no downloads and no external game engine — built on Next.js, Three.js/React Three Fiber, Rapier physics, Socket.IO, and PostgreSQL. Inspired by Deadshot.io, distinguished by unique hero abilities.

## Status

- **v1 — current stable build.** Live and playable at windarms.com. A complete browser multiplayer FPS: cinematic landing page, movement-shooter controller, authoritative multiplayer server, seven weapons, four maps, procedural audio, accounts/progression, anti-cheat, round-based matches, procedurally rigged characters. Full detail: [versions/v1.md](versions/v1.md).
- **v2 — current development target.** A ground-up visual/architecture rebuild centered on hero abilities (the four Founding Operators), a reworked matchmaking flow, and a new UI, set in an expanded "War Above The Storm" world. Not yet implemented. Full detail: [versions/v2.md](versions/v2.md).
